# YT Audio Pipeline

iPhone（iOSショートカット）からYouTubeのURLを送信し、PC上のローカルサーバーで音楽ファイルを自動ダウンロード・整理するためのAPIシステムです。
ダウンロードされた音楽はメタデータとジャケット画像が埋め込まれ、かつ音量がSpotifyなどの基準（-14 LUFS）に自動調整されるため、JellyfinやFeishin、Finampなどのツールで快適な音楽ライブラリを即座に構築できます。

## 主な機能

* **ダウンロード**: yt-dlpを使用し、最高音質の音声抽出、メタデータ（曲名・アーティスト）の付与、サムネイルの埋め込みを自動で行います。
* **音量の自動均一化（ノーマライズ）**: 
* **自動フォルダ整理**: ダウンロードした曲は「downloads/アーティスト名/曲名 [ID].m4a」という形式で自動的に分類保存されます。
* **サイレント常駐**: WindowsのVBScriptを利用し、PC起動時にターミナル画面を出さずにバックグラウンドで自動起動します。
* **自己修復**: VBScriptの起動時にyt-dlpを自動で最新版にアップデートし、YouTubeの仕様変更によるダウンロードエラーを未然に防ぎます。

## セットアップ手順

### 1. 必要なライブラリのインストール
Python 3.10以上をインストールし、プロジェクトのルートで以下のコマンドを実行して依存ライブラリを導入してください。

```bash
pip install -r requirements.txt
```

### 2. 自動起動の設定
プロジェクトフォルダ内の「start_server.vbs」のショートカットを作成し、Windowsのスタートアップフォルダ（shell:startup）に配置してください。

### 3. iOSショートカットの作成
assets/shortcut_01.pngを参考にして、iPhone側でショートカットを作成し、共有シートから実行できるように設定してください。

## 外出先からの利用について（Tailscaleの推奨）
自宅のWi-Fi環境外（モバイル通信など）からiOSショートカットを実行するためには、iPhoneとPCを安全な仮想ネットワークで接続する必要があります。
本プロジェクトでは、設定が非常に簡単でセキュアなVPN構築ツールである Tailscale の導入を強く推奨します。

### 1. インストール
PCとiPhoneの両方にTailscaleをインストールし、同じアカウントでログインします。

### 2. IPの変更
iOSショートカットの送信先IPアドレスを、ローカルIPではなくTailscaleが付与するIPアドレス（100.x.x.x等）に設定してください。
これにより、場所を問わずいつでもYouTubeから楽曲を追加できるようになります。

## Jellyfin・クライアント側の推奨設定
ライブラリの整合性を保つため、以下の設定を推奨します。

* **Nfoファイルの保存**: オフ
* **アートワークのフォルダ保存**: オフ
* **インターネットからのメタデータ取得**: オフ（※yt-dlpがファイル内に埋め込んだデータを優先させるためです）
* **イメージ取得（オーディオ）**: Image Extractorのみオン
